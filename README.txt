При запуске проекта можно выбрать в стартовом окне вид шифрования - RSA, Цезарь, Виженер или XOR.

Каждая из кнопок запускает отдельное окно для кажого вида шифрования. В каждом необходимо ввести сдвиг, ключ или данные для его рассчета и применить изменения.

Кнопки "копировать" помещают зашифрованную или расшифрованную строку в буфер обмена, чтобы было проще дешифровать и проверять корректность написанного кода.

Кнопки "рандом" задают случайное значение ключа или сдвига.
